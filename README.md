## Game Design Document (GDD)

### Game Name
Gravity Run

### Genre
2D Platformer (Gravity-Based)

### Elevator Pitch
Gravity Run is a 2D platformer where the player flips gravity with a single click to navigate parkour levels. The player decides whether to run on the ceiling or the floor — but spikes are everywhere. Reaching the finish line requires both fast reflexes and a sharp understanding of gravity mechanics.

### Core Loop
Run → Jump → Flip Gravity → Avoid Spikes → Reach the Finish

### Controls
- **W / A / S / D** → Move the character
- **Space** → Jump
- **Right Mouse Click** → Flip gravity

### Win Condition
Reach the finish point at the end of the parkour level.

### Art Style
Pixel Art

### Loss Condition
Touching the spikes results in instant death.
